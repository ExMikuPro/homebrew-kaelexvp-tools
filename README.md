# ExMikuPro Kaelexvp-tools

## How do I install these formulae?

`brew install exmikupro/kaelexvp-tools/<formula>`

Or `brew tap exmikupro/kaelexvp-tools` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "exmikupro/kaelexvp-tools"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
